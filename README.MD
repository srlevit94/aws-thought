## Description
This application the product of an EdX tutorial on AWS EC2 Instances and DynamoDB. The application allows users to write their thoughts, and even include a photo. 

## Technology Used
- Amazon Web Services
-- EC2 Instance
-- DynamoDB
- Express
- Multer
- JavaScript
